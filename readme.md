# sDAG-SDK-js - sDAGraph JavaScript API


## Installation
``` bash
npm i sdagsign
```
## Usage
```

const sdag = require("sdagsign")

const pri = "e7a3d2759c20e411516fe338bdd7121810dd131331550494bed4c80f070e621d"
let account = new sdag.Accounts.NewAccount(pri)

```
